## Frontmatter Reference

Required fields:
- `title`: The post title
- `date`: Publication date in YYYY-MM-DD format
- `description`: SEO description and preview text

Optional fields:
- `tags`: Array of tag strings for categorization
- `draft`: Boolean (true/false) - true posts won't appear in listings

## Writing Tips

1. **SEO**: Keep description under 160 characters
2. **Tags**: Use 3-5 relevant tags for better categorization
3. **Images**: Add alt text for accessibility
4. **Code**: Use language-specific syntax highlighting
5. **Links**: Test all external links before publishing

## Draft Workflow

1. Set `draft: true` to work on posts privately
2. Access draft posts directly via URL while developing
3. Set `draft: false` or remove field to publish
4. Published posts appear in listings and RSS feed
